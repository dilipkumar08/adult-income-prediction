Adult Income Prediction Model

Dataset: Adult Income Dataset

The dataset is used to predict whether an adult's income exceeds $50,000 per year. It contains various features including age, workclass, education, marital status, occupation, race, gender, capital gain, capital loss, hours per week, and native country. The target variable is income, which is binary, indicating whether the income is less than or equal to $50,000 or greater than $50,000.

Model: RandonForestClassifier

A random forest is a meta estimator that fits a number of decision tree classifiers on various sub-samples of the dataset and uses averaging to improve the predictive accuracy and control over-fitting.

Accuracy Score

The accuracy score of the RandomForestClassifier model on the test set is approximately 0.848.

The accuracy score of the DecisionTreeClassifier model on the test set is approximately 0.837.
